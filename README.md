# ONIXIA - Tienda de ropa online

## Descripción del proyecto
Este proyecto consiste en el desarrollo de un sitio web de tipo e-commerce llamado Onixia, una marca creada por mí con el objetivo de representar una identidad visual propia orientada a un diseño moderno y minimalista.

La idea principal de Onixia no es solo simular una tienda online, sino también funcionar como una base para futuros proyectos personales, donde pueda aplicar y demostrar mis conocimientos en desarrollo y diseño web. De esta forma, el sitio también se plantea como parte de mi portfolio profesional.

Además, el diseño visual del sitio fue acompañado por la creación de un logo propio, desarrollado especialmente para la marca Onixia, reforzando así su identidad y coherencia visual.

## Estructura
Para realizar este proyecto trabajé con las bases de HTML y CSS, entendiendo cómo se arma una página web desde cero. Organicé el contenido usando etiquetas semánticas como `<header>`, `<main>`, `<section>` y `<footer>`, separando cada parte del sitio de forma clara.

Utilicé CSS para darle estilo a la página y trabajé con Flexbox y Grid para ordenar los elementos en pantalla, logrando que las tarjetas se vean alineadas y organizadas.

## Tecnologías usadas
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript
- MockAPI (API REST)
- LocalStorage
- Google Fonts
- Font Awesome

## Funcionalidades
- Catálogo de productos dinámico desde API REST
- Carrito de compras persistente con localStorage
- Diseño responsivo para mobile y desktop
- Formulario de contacto funcional
- Sección de reseñas con Grid
- Menú hamburguesa animado

## Experiencia
Para la realización de este proyecto conté con una base previa en desarrollo front-end. Mi interés y formación en diseño gráfico me ayudaron mucho durante todo el proceso, pudiendo pensar el sitio desde lo visual y no solo desde lo técnico.

Me inspiré en referencias de Pinterest y utilicé el inspector del navegador para probar cambios en tiempo real y corregir errores. También utilicé herramientas de inteligencia artificial como apoyo en algunas partes, especialmente en CSS, pero siempre tratando de entender el código y adaptarlo a mi diseño.

El proyecto fue un proceso de prueba y error, donde fui aprendiendo, experimentando y ajustando detalles hasta llegar a un resultado que representa tanto mis conocimientos como mi forma de diseñar.
